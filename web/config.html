📋使用说明

  1. 在上方编辑你的模型配置和提示词内容
  2. 点击"生成 API 配置"和"生成提示词配置"
  3. 复制生成的YAML内容，分别粘贴到项目的 config/api_config.yaml 和 config/prompts_config.yaml
  4. 在"生成运行命令"区域选择参数，生成运行命令
  5. 在终端执行命令即可开始批量测试
