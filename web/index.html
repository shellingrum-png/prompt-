🚀 批量提示词测试系统

📖 使用步骤

1配置API密钥

打开 config/api_config.yaml 文件，填入你的API密钥

openai: api_key: "sk-..." base_url: "https://api.openai.com/v1" model: "gpt-4o" temperature: 0.7

2运行批量生成

在终端执行以下命令之一：

# 完整流水线（生成 + 评估） cd batch_test_system python3 main.py pipeline \ --input ../outputs/会话处理结果.xlsx \ --column "提示词Ainput" \ --prompt-type prompt_a \ --output results/full_results.xlsx # 仅批量生成提示词A python3 main.py run \ --input ../outputs/会话处理结果.xlsx \ --column "提示词Ainput" \ --prompt-type prompt_a \ --output results/prompt_a_results.xlsx # 仅批量生成提示词B python3 main.py run \ --input ../outputs/会话处理结果.xlsx \ --column "提示词Binput" \ --prompt-type prompt_b \ --output results/prompt_b_results.xlsx

3查看结果

生成的结果会保存在 results/ 目录下，包含：

  • 输入内容和生成的输出
  • 准确性、可读性、完整性三项评分
  • 评分理由和改进建议
  • 自动生成的统计报告（.txt）

💻 常用命令

cd batch_test_system && python3 main.py pipeline \ --input ../outputs/会话处理结果.xlsx \ --column "提示词Ainput" \ --prompt-type prompt_a \ --output results/prompt_a_evaluated.xlsx
cd batch_test_system && python3 main.py run \ --input ../outputs/会话处理结果.xlsx \ --column "提示词Binput" \ --prompt-type prompt_b \ --output results/prompt_b_results.xlsx
cd batch_test_system && python3 main.py evaluate \ --input results/prompt_a_results.xlsx \ --output results/prompt_a_evaluated.xlsx

💡 提示

  • 如需调整提示词内容，请编辑 config/prompts_config.yaml
  • 运行时使用 --delay 2.0 参数可增大调用间隔，避免限流
  • 自定义温度使用 --temperature 0.3 参数（数值越低越准确）
  • 评估模型可单独配置，使用更强的模型获得更准确的评分
